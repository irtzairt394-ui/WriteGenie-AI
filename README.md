# WriteGenie AI — Premium AI Writing Tools Platform

WriteGenie AI is a high-performance content generation platform powered by Google Gemini AI. It provides professional-grade tools for business emails, social media content, product descriptions, blog posts, ad copy, and high-quality rewriting.

## Features
- **Email Writer:** Persuasive, professional business emails.
- **Social Media Strategist:** Viral content for LinkedIn, Instagram, and more.
- **Product Descriptions:** Premium Amazon/Shopify-style copy.
- **Blog Content:** High-quality, SEO-optimized blog posts.
- **Ad Copy:** Performance-driven copy for Facebook and Google Ads.
- **Content Rewriter:** Elite-level editing and style transformation.

## Technology Stack
- **Backend:** Flask (Python)
- **Frontend:** Glassmorphism UI (HTML/CSS/JS)
- **AI Engine:** Google Gemini (Generative AI)
- **Deployment:** Docker on Hugging Face Spaces

## Deployment on Hugging Face
This space is configured using Docker. It binds to port `7860` as required by the Hugging Face platform.
